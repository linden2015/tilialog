# TiliaLog

A Java program that helps you log time.

![image](https://s27.postimg.org/5qz87modf/tilialog_0_10_reduced.png)

## Instructions

Run as follows:

    cd output
    java com/tilialog/App &

## Requirements

* Java 1.8+
